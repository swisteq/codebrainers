# Selenium + Pytest Automation Tests

Projekt zawiera automatyczne testy UI napisane w Pythonie z użyciem:

- Selenium WebDriver
- Pytest
- Chrome WebDriver

Testy obejmują weryfikację stron:
- https://codebrainers.pl/
- https://opensource-demo.orangehrmlive.com/
